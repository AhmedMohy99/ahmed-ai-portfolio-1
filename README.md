# Ahmed Mohy AI Portfolio Website

A clean, responsive portfolio website for showcasing AI and machine learning projects.

## Files
- `index.html` → main page
- `style.css` → styling
- `script.js` → simple JavaScript
- `vercel.json` → Vercel configuration

## Deploy to Vercel

### Option 1: Drag and drop
1. Put all files in one folder.
2. Zip the folder if needed.
3. Go to Vercel.
4. Create a new project.
5. Upload the files or import from GitHub.

### Option 2: GitHub + Vercel
1. Create a new GitHub repository.
2. Upload these files.
3. Push the repo.
4. Connect GitHub to Vercel.
5. Import the repository.
6. Click Deploy.

## Customize
Update the project links in `index.html` when each GitHub repo and demo is ready.
